# 📊 The Impact of Infrastructure Disruptions on Students’ Academic Performance in Sri Lanka

A survey-based academic research project investigating the impact of infrastructure disruptions on students’ academic performance in Sri Lanka. Data analysis and visualization were supported using Excel and Power BI.

## 🎯 Project Overview

This project involved transforming raw survey data into meaningful insights through data preparation, analysis, and interactive dashboard visualization.

## 🛠 My Contributions

### Data Preparation (Primary Role)

- Converted raw CSV files into structured datasets
- Cleaned missing, inconsistent, and error values
- Performed manual thematic analysis (Manually, without AI)
- Prepared analysis-ready datasets using Power Query and Excel
- Applied formulas, data validation, and conditional formatting to ensure data quality
